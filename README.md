# Game Night Quiz

Un quiz interattivo per serate tra amici con domande di cultura generale.

## Caratteristiche

- 10 domande di cultura generale (espandibili a 60)
- Sistema di punteggio in tempo reale
- Classifica live
- Supporto per immagini, audio e video
- Design moderno e responsive

## Come usare

1. Apri il sito web
2. Inserisci il tuo nome
3. Rispondi alle domande prima che scada il tempo
4. Competi con gli altri giocatori per la prima posizione in classifica

## Tecnologie

- HTML5
- CSS3
- JavaScript (ES6+)
